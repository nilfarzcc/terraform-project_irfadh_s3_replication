# Terraform Project: S3 Replication

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/nilfarzcc/terraform-project_irfadh_s3_replication.git

2. Go inside the project folder:
   cd terraform-project_irfadh_s3_replication

3. Initialize Terraform and apply:
   terraform init
   terraform plan
   terraform apply

Note: The s3_bucket module is automatically downloaded by Terraform and stored in .terraform/modules/.
